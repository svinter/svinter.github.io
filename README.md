# stevevinter.com

Personal website for Steve Vinter, hosted on GitHub Pages.

## Structure

```
/                   — Home
/whatido/           — What I do
/photography/       — Photography (links to SmugMug)
/resources/         — Books, articles, notes
/schedule/          — Calendly embed
/about/             — Biography

/recipes/           — Private: family recipes
/library/           — Generated by Libby (Dashy module)
  /books/
  /articles/
  /tools/
```

## Authoring

All pages are plain HTML files sharing a single stylesheet at `/css/style.css`.

- **Public pages**: full nav bar
- **Private pages** (`/recipes/`, `/library/`): minimal nav (name only, no links)

To add a recipe: copy an existing recipe page, update the content, add a link to `/recipes/index.html`.

## Fonts

Google Fonts: Cormorant Garamond (headings) + Inter (body). Loaded via `@import` in `style.css`.

## Deployment

Push to `main` branch → GitHub Pages deploys automatically.

Custom domain: `stevevinter.com` — DNS configured via registrar to point to GitHub Pages.

## Calendly

Update the iframe `src` in `/schedule/index.html` with your Calendly URL.

## Libby integration

The `/library/` directory is managed by the Libby module in Dashy. The `make` action generates HTML files and pushes them here automatically.
